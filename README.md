## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Help command

```bash
# create a project nest
$ nest new projectName

# create a new module users example
$ nest g module users

# create a new controller for the module users
$ nest g controller users

# create a new service for the module users
$ nest g service users

# create a new resource
$ nest g resource users

```
